== Description ==

= Connect your blog with SpeedBlogging system = 

SpeedBlogging is an innovative software created by Chris Record that allows you to take any sources such as images, Facebook posts, Pinterest pins and many more other sources and turn them into a Wordpres post or Wordpress list post.

This plugin will enable your blog to connect with your account at SpeedBlogging so you can utilize this system

== Installation ==

1. Backup your Wordpress site and database
2. Upload the content inside the zip file to '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in Wordpress
4. Visit SB Connector for scanning and options

== Frequently Asked Questions ==

= How can I use this plugin? =
* Simply generate the connector key in SB Connector. Then, in your SpeedBlogging site add a new blog using this Connector key and API Endpoint
